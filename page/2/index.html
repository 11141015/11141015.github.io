bb我没睡着，想了很多很多。不想忘记这些思绪，写完这些再睡。今天这个更新算作第五天的更新吧。

刚刚洗完澡之后我有偷偷翻你的微博小号看，但我没看具体内容，就只是一直往下翻，然后看到你是三月初才又开始重新追星的，在这之前都没有。这也是我们感情变淡的时间点吧，那时候我很情绪化，还很自我，根本没有去顾及你的感受。后面越来越离谱，有一天晚上喝了酒之后还跟你说觉得你胖了，现在回想起来真的太过分了。bb那时候应该很受伤吧，对不起。虽然这几天我已经跟你说过无数次对不起了，但我还是要在这里再说一次，对不起，我真的不应该让陪伴自己的女生受到这样的伤害，而且还是来自于我的伤害。

我也想明白了，你现在就想要自己一个人，好好找工作好好追星，我会尊重你的选择。你好好工作，我也好好学习，等你工作稳定了之后我们可以再一起去吃好吃的东西、看好看的日落。谢谢bb关心我的眼睛，我会保重身体和好好学习的。一起加油，我们一定会在更高处见。嗷对，今晚我很难过，跟你说我觉得这个博客以后可能更不下去了。但我刚刚发呆的时候在想，其实继续更也挺好的，就好像你依旧陪在我的身边，只不过是换了一种方式而已。所以我决定继续更，尽量日更，真的有事的时候再请假，然后会一直更到你有新男朋友。如果到域名快要过期了还在更，那我就会续多一年，如此往复。

嗷，还有，你以后能不能多发点微博，因为这可能是我唯一能了解到你生活的方式了。如果实在没什么发的就不发吧，以后找你出去玩的时候再问你。

希望我们都能快速健康的成长，以及快快在未来再遇见！

晚安bb

分手 Day 4

时间过的蛮快的，第四天了。现在是下午两点，在Science Library里看pdf，休息时间用来更新一下失恋日记。

昨天买了一瓶新的洗发水，买回家之后突然发现跟你的味道满像的(？也可能是错觉)。要给你的东西还没收，一会回家午睡的时候收。今晚约了你吃饭还有去海边，希望今晚相处的轻松自然。嗷，今天去系里问了一下TA的事情，说是已经招了别人了，有点sad。还有前些天东哥跟我说了一个代写作业的兼职，不过那时候刚分开很难过，就跟他说之后再说。过几天我再问问东哥。

嗷，我觉得东海岸公园光污染过于严重，晚上带你去我以前会去坐着发呆的海边。一会回家还要记得去fair price买瓶Jinro，不知道你喝不喝，不喝到时候我就自己喝，不过感觉喝一整瓶太多了。继续看pdf了，看多一会回家了，晚上见。

分手 Day 3

Posted on 2021-05-02 | | Views:

梦到你了，梦到我们在打王者(？感觉很奇怪)，我们是不同队的，你跟我说输了就别再找你了。游戏里你玩的很厉害，总是能找到我的破绽，还好后来我赢了。然后我就醒了，睡不着了，凌晨五点多。躺了一会还是睡不着就爬起来了。不知道为什么，洗漱的时候突然想起跟你在Science一起吃麻辣香锅的画面。洗漱完看着空空的宿舍，想起了很多你在宿舍里的画面，也才反应过来，嗷，你不会再来了。

梦里好像还有一段一起生活的画面。梦里我们一起租了一间HDB，下了班会一起做饭，很开心。半梦半醒的时候我在想，跟眼前的这个人结婚可以吗，想了几秒，觉得可以，然后就跳到第一段写的那个梦了(？)。

也许还会在坡待很久吧，你也是吧。我自己一个人活的满辛苦的，周围的一切还都带着你的影子。对了，你什么时候搬家哦，想问你下周二有没有时间去东海岸玩，等明天你醒了再问你吧。嗷，还有等你换了新的房子之后，就没有我的影子了，但你不要忘记我哦，因为我还在这里用我自己的方式继续爱着你。

你这几天过的好吗？我好想你。

分手 Day 2

Posted on 2021-05-01 | | Views:

昨天可能是因为大家基本上都考完试了，我在图书馆自习的时候周围没什么人。本来听着白噪音学习挺好的，后来学了两个小时想着听首歌放松一下，歌单里面随机播放了一首《我还是爱着你》，直接就给我整emo了。也没再切歌，就一直循环这歌学习，学到后面无比难受，回家路上去超市买了酒。回家之后把《我还是爱着你》弹会了，和弦挺容易的，然后开始跟PUA的王边打游戏边喝，打到早晨睡觉。醒了之后喝了很多杯热水，但头还是很晕，人也不是很清醒，挺好的，至少不会去想太多感情的事。干，现在都八点多了，我头还是晕的。

昨晚睡觉的时候半梦半醒，在想你留了什么东西在我这，我留了什么东西在你那，然后发现，我好像也没送过你啥东西。花始终都会谢，以后还是得送点能保存的东西，这样还能承载一部分关于我的记忆。那条红丝带还在床下面，明天再捡出来洗，今天太累了。还有那个发绳也好久没戴过了，不记得上一次出去玩是什么时候了。那个支架不知道要不要还给你，因为你已经有一个支架了。我有一个纸盒可以代替它，不还的话我就收起来吧，也不用它了。床单还没收，前天洗的衣服还铺在上面，明天再收吧。分这些东西的时候感觉像是又分了一次手，挺难受的。

不知道说分手的人会不会难过，反正我是挺难过的。今天一天都晕晕的，也没干啥，先这样吧，明天再写。最后放一个昨晚搞的《我还是爱着你》的翻唱，有的地方没唱上去，不过我录了两轨，有一轨专门弹了旋律，能凑合听。
