My Last Letter

Posted on 2020-07-02 | | Views:

给我爱的你：

本来想视频或者打电话跟你说这些话，但我不知道要怎么开口，所以就把它们写了下来。

虽然我们在一起的时间很短，但这些天我过的都很开心，谢谢你让我爱你。每次偷偷给你买东西的时候我都很开心，一想到你收到惊喜的样子，就会不自觉的笑。我吉他弹的垃圾，但每次弹唱给你听，你都会说好听。还有刚认识的时候，我们总是通宵打电话，那时候的开心也是真实的。总之，很谢谢你走进我的生活，并带给了我很多快乐。

后来我开始想象我们的未来，想我们未来的样子。我们也交流了很多这方面的想法。但很可惜，好像真的没有一个两全的办法，我们的未来好像很遥远，又很模糊。

我很喜欢你，直到现在我在写这些话的时候，也还是很喜欢你。但我们都是成年人了，有很多现实的东西需要去面对。比如，我们没有什么感情基础，再加上这个疫情，不知道要到什么时候才可以见面。即使没有疫情，由于我们的经济能力不大行，我们最多也就两个月见一次面。如果顺利的话，我明年下半年就要去美帝读书了。去美帝之后见面就更加不容易了。而再之后的未来也是模糊不清，结束异地遥遥无期，没有一个盼头。

所以不如就先分开吧，倘若日后我从美帝回来，我们还有缘的话，就再续前缘。

我没有玩弄你的感情，我自己也是真实的难受。但长痛不如短痛，如果等到几年之后再分开的话，我们要承受的伤心和难过只会更多。

那就有缘再见啦

小王
